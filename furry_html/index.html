Photoshop Actions Paillettes

Content:

  • 3 .atn files (action), 12px, 20px and 30 px diameter.
    • 5 actions x each .atn file
    • 15 total actions
  • 1 .abr file (Paillettes brushes)
  • 1 .asl file (Paillettes style)
  • 1 Help File
  • 15 Photoshop files to practice
  • 10 tutorial videos links

Install:

Load the Paillettes Actions

  • ANIPAILLETTESØ12.atn
  • ANIPAILLETTESØ20.atn
  • ANIPAILLETTESØ30.atn

Load the Paillettes Brushes

  • PailletteBrushes.abr

Load the Paillettes Styles

  • PailletteStyles.asl

Easy to use:

  • Select the layer where the action will run
  • Set the colors
  • Choose the action from the Actions Panel
  • Play the action
  • Play the animation by clicking the play button at the Timeline Panel
  • Recommended minimum file size 1000 x 1000 px

Optional

  • Change the paillettes colors
  • Show or hide the sparkles
