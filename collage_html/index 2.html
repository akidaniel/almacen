Install:

Load any or all the Vertical Collage Actions

  • XX_VERTICAL_COLLAGE_2_LARGE.atn
  • XX_VERTICAL_COLLAGE_2_SMALL.atn

Load the seam pattern brushes

  • SEAM_PATTERN_2.abr

Custom:

  • The action ends with a Group named as the action played and contains:
    • Text Layers Group
    • Image Layers Group
  • Transform or move the content of a layer unlocking the mask by clicking the Link to the mask icon.
    • Click the Layer Thumbnail to move or transform the layer content
    • Click the Mask Thumbnail to move vertically the seam pattern
  • Hide the Layer Style to remove the pattern stroke or double click the Layer Style to edit the stroke and change its width or color.
  • The font of the caption is a system font: Arial Bold. Edit the text to change the font and/or the content or hide the Text Layer’s Group if it’s not necessary to display them.

Instructions:

  • Make a new file with the final document size. Any height is posible.
  • Go to the menu View → Rulers and from the left ruler drag a vertical guide until the document center
  • Distribute the images that will make up the collage in the upper layers by copy-pasting from other open documents or placing them from the menu File → Place Embedded.
  • Move the layer content until the main object is vertically centered in the document

  • Transform the layer to adjust the content size and position

  • Choose the action according to the number of images the final collage will have
  • Run the action
    • The layer content height must be equal or larger than the document height, any width
    • The layer content can be rotated if the main object is not vertically aligned or just to adjust its position
    • Move the layer until the main object is vertically centered in the document. The action will use the central axis to relocate each image to the corresponding position in the collage.
