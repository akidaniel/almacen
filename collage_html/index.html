Vertical collage

  • Actions to create vertical collages of 5, 6, 7, 8, 9 and 10 images with five different seam designs

  • From 5 to 10 images
  • x 2 different sizes:
    • Small: pattern width = 100px
    • Large: pattern width = 200px
  • = 50 total actions
  • Non-destructive: keeps the original images untouched
  • Well arranged masked image layers
  • Editable Layer Style to change the color, width or hide the pattern stroke
  • Separated Caption Text Group Layers

Instructions

  • Make a new file with the final document size. Any height is posible.
  • Go to the menu View → Rulers and from the left ruler drag a vertical guide until the document center
  • Distribute the images that will make up the collage in the upper layers by copy-pasting from other open documents or placing them from the menu File → Place Embedded.
  • Move the layer content until the main object is vertically centered in the document

  • Transform the layer to adjust the content size and position:
    • The layer content height must be equal or larger than the document height, any width
    • The layer content can be rotated if the main object is not vertically aligned or just to adjust its position
    • Move the layer until the main object is vertically centered in the document. The action will use the central axis to relocate each image to the corresponding position in the collage.

Load the Mosaic Tiles Brushes

  • TILES_03.abr

Easy to use:

  • Open the image file where the action will run
  • Choose the action from the Actions Panel
  • Play the action
  • The action ends with a Group named as the action played and contains:
    • Sparkle Layers Group
    • Light Mosaic Layers Group
    • Mosaic Layers Group
    • Mosaic Background Layer
  • From the Timeline Panel, select the Frame #1 and play the animation
  • Save just the image or export the animation as a .gif format

Optional:

  • Change the background color
  • From the Timeline Panel, select the Frame #1 and show or hide the Spakles Group Folder to see the animation with or without sparkles
