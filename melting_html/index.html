Melting Pictures Photoshop Actions

Content:

  • 1 MELTING_PICTURES_ACTIONS.atn file with 4 actions

  • 1 MELTING_PICTURES_BRUSHES.abr file

  • 1 PDF Help File: English-Español

  • 1 tutorial video link

Install:

Load the Melting Pictures Actions

  • MELTING_PICTURE_ACTIONS.atn

Load the Melting Pictures Brushes

  • MELTING_PICTURE_BRUSHES.abr

Easy to use:

  • Open the image file

  • Check the size from the menu Image > Image Size

  • Choose the action from the Actions Panel. There are four actions for different images sizes:

    • MELTING_IMAGE_1000 for images ≥600 / ≤1500 pixels wide

    • MELTING_IMAGE_2000 for images ≥1200 / ≤2500 pixels wide

    • MELTING_IMAGE_3000 for images ≥2200 / ≤3500 pixels wide

    • MELTING_IMAGE_4000 for images ≥3200 / ≤4500 pixels wide

  • Play the action

  • The action ends with a New Layer Group named Melting Image containing:

    • A PAINTFLOW group

    • A PAINTDROPS Group

    • A MAINIMAGE Group

    • A background Solid Color Layer
