Animated Mosaic Actions

Content:

  • 1 FURRY_ACTIONS.atn file with 32 fur texture actions + 9 pets faces actions
  • 1 FURRY_BRUSHES.abr file
  • 1 FURRY_STYLE.asl file
  • 1 FURRY_SHAPES.asl file
  • 1 PDF Help File: English-Español
  • 9 Photoshop files to practice
  • 1 tutorial video link

Install:

Load the FurryPet Actions

  • FURRY_ACTIONS.atn

Load the FurryPet Brushes

  • FURRY_BRUSHES.abr

Load the FurryPet Styles

  • FURRY_STYLE.asl

Load the FurryPet Shapes

  • FURRY_SHAPES.asl

Easy to use:

  • Select the layer where the action will run
  • Set the foreground and background color for the dark and light hair
  • Choose the fur action from the Actions Panel. They are listed by name in alphabetical order from 01 to 32
  • Play the action
  • The action ends with a New Layer named as the action played
  • Recommended minimum file size 1000 x 1000 px

Optional, use the pet face actions:

  • Make a selection in the middle of the face location
  • Select one of the nine face actions located at the bottom of the fur-actions list
  • Play the action
  • The action ends with a New Group named as the action played
    • Select this face group to scale it or move it
  • The Face Group contents other groups or vector shape layers with the face parts: eyes, nose, ears, mustache
    • Select each shape layer to transform
    • Click the layer eye icon to show/hide the content
    • Click twice the layer thumbnail to edit the content and change the color
