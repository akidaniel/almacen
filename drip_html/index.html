Drip Photo

  • Easy to use: open the image file and play the action
  • Non-destructive: keeps the original image untouched
  • Dripped edges separated into 2 different easily customizable Color Fill Layers:
    • Upper Edge
    • Lower Edge
  • 6 different drip styles

  • x 2 drip thicknesses:
    • 40 px best suited for images from 1500 to 3500 px wide
    • 80 px best suited for images from 3500 to 6000 px wide


  • = 12 total actions
  • Works with RGB or CMYK
  • Photoshop versions CS6, CC, CC2014, CC2015, CC2017, CC2018, CC2019+

Content:

  • DRIP_PHOTO.atn file with 12 total actions
  • DRIP_PHOTO.abr file
  • PDF Help File

Install:

Load the Drip Action file

  • DRIP_PHOTO.atn

Load the Drip Brushes

  • DRIP_PHOTO.abr

Running:

  • Open the image file
  • Click the image layer
  • Choose the action from the Actions Panel
  • Play the action
  • The action ends in a new document with a Layer Group named as the played action containing the following from bottom to top:
    • BASE: the background image
    • LOWER_BORDER: a Layer group with:
      • LOWER_BACKGROUND: the image background extension
      • LOWER_EDGE:an editable Color Fill Layer with the bottom dripping edge in dark gray #181818
    • UPPER_BORDER: a Layer group with:
      • UPPER_BACKGROUND: the image background extension
      • LOWER_EDGE:an editable Color Fill Layer with the upper dripping edge in dark gray #181818

Customization:

Show / Hide:

Click on the lower or upper Edges Layers Group eye icon to show / hide each drip or show both:

Position:

Using the Move Tool, reposition the upper or lower border layer to adjust its location:

Colors:

Double click on the LOWER_BORDER layer thumbnail to change the bottom drips color:

Double click on the UPPER_BORDER layer thumbnail to change the upper drips color:
