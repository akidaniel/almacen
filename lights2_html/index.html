  • The group of brushes: LIGHTS.abr
  • The help and instructions file, English/Spanish:
    Bicolor Fairy Lights Actions Instructions.pdf

Installing:

Load the Bicolor Fairy Lights Actions

  • BICOLOR_FAIRY_LIGTHS.atn

Load the Bicolor Fairy Lights Brushes

  • LIGHTS.abr

Use:

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Open the image file
  • From the Layers Panel, add a new layer and fill in the area where the lights will be using the brush or the selection tools and hide this layer by clicking on the eye icon
  • From the Tools Panel, choose the foreground and background colors for the colors of the lights
  • Go to the Actions Panel and play the LIGHTS action
  • The action ends with four New Groups containing Color Fill masked layers with the lights and a Color Fill Layer with the cable

Animation

  • Once the LIGHTS action is done, select one of the tree Animation Actions and play it:
    • Blink
    • Fading
    • Soft Fading

Optional:

  • Show or hide each layer with the light components
  • Edit the Color Fill layers to change the color

Minimum recommended file size: 1500px width

Works with versions CS6, CC, CC2014, CC2015, CC2017, CC2018, CC2019
