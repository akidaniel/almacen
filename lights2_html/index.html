    Multicolor Fairy Lights Actions Instructions.pdf

Installing:

Load the Multicolor Fairy Lights Actions

  • MULTICOLOR_FAIRY_LIGTHS.atn

Load the Multicolor Fairy Lights Brushes

  • MULTICOLOR_LIGHTS.abr

Use:

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Open the image file
  • From the Layers Panel, add a new layer and fill in the area where the lights will be using the brush or the selection tools and hide this layer by clicking on the eye icon
  • From the Tools Panel, choose the foreground and background colors for the colors of the lights
  • Go to the Actions Panel and play one of the three MULTICOLOR LIGHTS action
    • MULTICOLOR_LIGHTS
    • MULTICOLOR_LIGHTS_COMPRESSED
    • MULTICOLOR_COMPACT
  • The action ends with a New Group containing the Color Groups with Color Fill masked layers with the lights and a Color Fill Layer with the cable

Animation

  • Once the LIGHTS action is done, select one of the five Animation Actions and play it:
    • Flash
    • Blink
    • Fading
    • Bicolor Fading
    • Tricolor Fading

Optional:

  • Show or hide each layer with the light components
  • Edit the Color Fill layers to change the color

Minimum recommended file size: 1500px width

Works with versions CS6, CC, CC2014, CC2015, CC2017, CC2018, CC2019
