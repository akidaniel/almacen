Content:

  • The actions set file: NEONSIGN_ANIMATION.atn
  • The help and instructions file: Neon Sign Actions Help.pdf

Installing:

Load the Neon Sign Animation Actions

  • NEONSING_ANIMATION.atn

Instructions:

  • Click the layer where the action will run.
    It can be:
    • Text Layer
    • Isolated image (pixel layer)
    • Shape Layer
    • Smart Object
    • Group of layers
  • Choose a foreground color for the neon body ⇢ outer color
  • Choose a background color for the neon highlight ⇢ inner color
  • Go to the Actions Panel and play a NEON ANIMATION action:
    • NEON ANIMATION SIMPLE
      • Suitable for a single word, one line text or layers with rectangular landscape format content
      • Recommended file width size: any width in a 7:1 aspect ratio
      • Recommended file heigth size: from 500 to 1500 px
    • NEON ANIMATION DOUBLE
      • Suitable for two-line words or layers with square proportion content
  • The action ends with a New Group containing six different animations

Animation

  • Once the NEON ANIMATION action is done:
    • Timeline Panel ⇢ Click the Frame#1
    • Click the Play button to see the animation
  • To play and see other animation:
    • Timeline Panel ⇢ Click the Frame#1
    • Layers Panel ⇢ Show or hide the animation group
    • Timeline Panel ⇢ Click the Play button

Features:

  • NEON ANIMATION SIMPLE
    • 8 neon lines
    • 18 animation frames
  • NEON ANIMATION DOUBLE
    • 14 neon lines
    • 30 animation frames
  • Transparent background

Recommended layer content size for best result:

  • Width: Any size
  • Height: from 500 to 1500 px

Works with versions: CS6, CC, CC2014, CC2015, CC2017, CC2019 (v.)*, CC2020, CC2021
