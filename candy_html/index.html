Installing:

Load the Candy Actions

  • CANDY_ACTIONS_BIG.atn
  • CANDY_ACTIONS_MEDIUM.atn
  • CANDY_ACTIONS_SMALL.atn

Load the Candy Brushes

  • CANDY_BRUSHES.abr

Use:

Multicolor Actions

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Open the image file
  • From the Layers Panel click the layer where the candies will be
  • Go to the Actions Panel and play one of the Multicolor Candy actions
  • The action ends with a new group containing the groups of candies separated by color with masked Color Fill Layers and a background Color Fill Layer with the base object

Bicolor Actions

  • From the Tools Panel choose the foreground and background colors for the candies
  • Follow the same steps as in Multicolor actions

Result:

Well-organized result grouped by color with masked Color Fill layers allowing to show, hide, move the position and change the colors by editing each layer with a double click

Works with:

  • Pixel Layers
  • Vector Shape Layers
  • Text Layers
  • Smart Objects
  • Layer's Group Folders
  • Photoshop versions CS6, CC, CC2014, CC2015, CC2017, CC2018, CC2019
  • Minimum recommended file size: 1000px width

Font used in the text example

  • Magnolia Script
