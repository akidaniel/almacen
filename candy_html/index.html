Installing:

Load the Candy Actions

  • CANDY_ACTIONS_BIG.atn
  • CANDY_ACTIONS_MEDIUM.atn
  • CANDY_ACTIONS_SMALL.atn

Load the Candy Brushes

  • CANDY_BRUSHES.abr

Use:

Multicolor Actions

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Open the image file
  • From the Layers Panel click the layer where the candies will be
  • Go to the Actions Panel and play one of the Multicolor Candy action
  • The action ends with a New Group containing the Candy's groups by color and Color Fill masked layers with the candies and a Color Fill Layer with background object

Bicolor Actions

  • From the Tools Panel choose the foreground and background colors for the candies
  • Follow the same steps as in Multicolor actions
