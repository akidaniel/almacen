Cube Explode Photo Effect

3d mapping effect of an image on a cubes perspective with just a smart object replacement.

  • Layered PSD File
  • Smart Object Replacement
  • 2800px X 2800px / 72 ppp
  • RGB
  • Well Organized Layer Groups
  • Background in a separated Layer Group
  • Editable background:
    • Change the color
    • Transparent (hide it)
  • Back Glow in a separated Layer to show or hide
  • Edge Shine in a separated Layer Group to edit, show or hide
  • Minimum Photoshop version: CS4
  • PDF help and instructions
  • Video tutorial link

Download Content:

  • 1 PSD File
  • PDF help file with a tutorial video link

Instructions:

  • Open the Cube Explode.psd file
  • Edit the Smart Object
  • Place or paste the new image
  • Adjust, scale, move or rotate the new image layer
  • Save the Smart Object
  • See the result in the original Cube Explode file
  • Make any porrible customization
  • Save your file as a new document

Customization:

  • Show or hide the Edge Shine
  • Show or hide the Back Glow
  • Transparent Background hiding the background group
  • Change the Background Color
