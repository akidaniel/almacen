Content:

  • The action file: POP COVER CIRCLES.atn
      With two animation styles:
    • ANIMATION ALL AT ONCE
    • ANIMATION BEATING
  • The help and instructions file, English/Spanish:
    Pop Cover Circle Action Instructions.pdf

Install:

Load the Pop Cover Circle Actions

  • POP COVER CIRCLES.atn

How it works:

  • From the Layers Panel click the image layer
  • From the Actions Panel select the POP COVER CIRCLES action
  • Press the Play button ►

Output:

  • The action ends in a new file with a group containing all the layers of the Pop Cover Circle effect (the original file remains intact)
  • The animation has 32 frames where frame 1 and 17 are the original image and the image with the old effect, depending on the chosen animation style
  • From the Timeline Panel, click on frame 1 and press the Play Button ► to see the animation
  • Save the file in GIF format if you want to keep the animation
  • From the Timeline Panel, click on the frame with the image old effect (1 or 17 depending on the animation created) and save the file as .jpg or .png to preserve it as a static image

Features:

  • Transparent background
  • All the rays in editable Vector layers
  • All rays well arranged in separated Layer Groups
  • 30 animation frames ready to save as a GIF format
  • Recommended minimum image size: 800px x 800px
  • Recommended maximum image size: no limits!
