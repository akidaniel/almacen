Content:

  • The action file: POP COVER CIRCLES.atn with the action POP COVER CIRCLES + 2 animation actions:
    • ANIMATION ALL AT ONCE
    • ANIMATION BEATING
  • The help and instructions file, English/Spanish:
    Pop Cover Circle Action Instructions.pdf

Install:

Load the Pop Cover Circle Actions

  • POP COVER CIRCLES.atn

How it works:

  • From the Layers Panel click the image layer
  • Clip the image leaving visible only the object from where the POP COVER CIRCLES will emerge
  • From the Tools Panel choose a foreground and a background color for the rays, the action will distribute them over each element
  • From the Actions Panel select the POP COVER CIRCLES action
  • Press the Play Action button ►

Output:

  • The action ends in a new file with four groups containing the Pop Cover Circle layers, a background layer and a top layer with the main object (the original file remains intact)
  • All the rays are Shape Layers grouped according to their size and position in the image:
    • Small Rays
    • Medium Rays
    • Big Rays
    • Exterior Rays

Animation:

  • From the Actions Panel, select one of the two animation actions:
    • ANIMATION ALL AT ONCE
    • ANIMATION BEATING
  • Press the Play Action button ►
  • From the Timeline Panel, click on frame #1 and press the Play Animation button ► to see the animation
  • Save the file in GIF format to keep the animation

Features:

  • Transparent background
  • All the rays in editable Vector Layers
  • All rays well arranged in separated Layer Groups
  • 30 animation frames ready to save as a GIF format
  • Any size! Recommended minimum image size: 800px x 800px
