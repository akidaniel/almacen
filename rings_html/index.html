Concentric Rings

  • Actions to create concentric rings background from the image color
  • 3 actions for three ring thicknesses:
    • Fine
    • Medium
    • Coarse
  • Just 1 click
  • Non-destructive: keeps the original images untouched
  • Well-organized layers in groups
  • Background rings with Adjustment Layers to modify or change its color
  • Transparent background
  • Works with any image size. Recommended minimum file size: 1200 px
  • Works with RGB or CMYK
  • After playing an action, select the mask layer again to run the next one. The result of the new action will be automatically placed at the top of the layer panel
  • Photoshop versions CS6, CC, CC2014, CC2015, CC2017, CC2018, CC2019+

Content:

  • 3 action files
    • CONCENTRIC_RING_FINE.atn
    • CONCENTRIC_RING_MEDIUM.atn
    • CONCENTRIC_RING_COARSE.atn
  • 1 PDF Help File
  • 1 tutorial video link

Install:

Load the Concentric Rings Actions

  • CONCENTRIC_RING_FINE.atn
  • CONCENTRIC_RING_MEDIUM.atn
  • CONCENTRIC_RING_COARSE.atn

Instructions:

  • Open the image file
  • Mask the main picture area
  • From the Actions Panel choose the action
  • Click the Play Action button at the panel bottom
  • The action ends with a New Layer Group named as the action played

Custom:

  • Show or hide the Left or Right group of layers by clicking the layer’s eye icon to display only the half of the background
  • Open the Left and/or Right layer group to show its content by clicking the left icon of the layer thumbnail.
  • Modify the adjustments layer’s parameters by double clicking the adjustment layer thumbnail to show the options panel. Move the sliders to change the background rings color
  • To play another action just click the Mask Layer
