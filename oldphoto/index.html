Old Photo Maker

Content:

  • OLDPHOTO.atn
  • OLDPHOTO.abr file

Install:

Load the Old Photo Action

  • OLDPHOTO.atn

Load the Old Photo Brushes

  • OLDPHOTO.abr

Use:

  • Set the Brush Tool to normal Mode, 100% opacity
  • Click and hide the image layer
  • Go to the Actions Panel and play play the OLDPHOTO action
  • The action ends with a New Group masked with the effects layers

Optional:

Invert the resulting mask to change the wipe direction
