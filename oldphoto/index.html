Content:

  • The action file: OLDPHOTO.atn
      Containing the two animation styles:
    • PHOTO_NEW2OLD
    • PHOTO_OLD2NEW
  • The group of brushes: OLDPHOTO.abr
  • The help and instructions file, English/Spanish:
    Damaged Photo Action Instructions.pdf

Install:

Load the Old Photo Action

  • OLDPHOTO.atn

Load the Old Photo Brushes

  • OLDPHOTO.abr

How it works:

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Click the image layer
  • From the Actions Panel select one of the OLDPHOTO action
    • PHOTO_NEW2OLD to get an animation from the original photo to the old one
    • PHOTO_OLD2NEW to get an animation from the old photo-effect to the original
  • Press the Play button ►

Output:

  • The action ends in a new file with a group containing all the layers of the old photo effect (the original file remains intact)
  • The animation has 32 frames where frame 1 and 17 are the original image and the image with the old effect, depending on the chosen animation style
  • From the Timeline Panel, click on frame 1 and press the Play Button ► to see the animation
  • Save the file in GIF format if you want to keep the animation
  • From the Timeline Panel, click on the frame with the image old effect (1 or 17 depending on the animation created) and save the file as .jpg or .png to preserve it as a static image

Features:

  • Random results
  • Transparent background
  • All effects in separated layers
  • Most effects in editable Color Fill layers
  • Effects with fine details
  • File ready to save as a GIF format with 32 animation frames

Works with:

  • Any color mode: RGB, CMYK, Grayscale...
  • Any orientation: landscape, square or portrait
  • Image layers
  • Clipped images
  • Vector object layers
  • Text layers
  • Group layers
  • Smart Objects
