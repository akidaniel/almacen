Content:

  • The action file: OLDPHOTO.atn
  • The group of brushes: OLDPHOTO.abr

Install:

Load the Old Photo Action

  • OLDPHOTO.atn

Load the Old Photo Brushes

  • OLDPHOTO.abr

Use:

  • Select the the Brush Tool and make sure at the top options it's in a normal Mode, 100% opacity
  • Click and hide the image layer
  • Go to the Actions Panel and play the OLDPHOTO action
  • The action ends with a New Group masked containing the effects layers

Optional:

  • Show or hide the effects layers
  • Edit each Color Fill layer to change the color
