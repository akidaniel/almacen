Retro Mosaic

  • Easy to use: just 1 Click actions!
  • No mask or selection needed: open the image and play the action
  • Non-destructive, keeps the original images untouched
  • 12 different tile designs grouped in the same action
  • 5 tile sizes: 20, 30, 40, 50 and 60 px
  • 5 total actions
  • Custom color scheme
  • The twelve tile designs well arranged in layer groups
  • Customizable after playing the action
    • Change or modify any color
    • Show or hide the bevel effect
  • Excellent final quality
  • Any image size (minimum 300px X 300px)
  • RGB or CMYK
  • Works with Pixel Layers, Layer Group, Smart Objects, Shape layers or Text layers

Content:

  • 1 RETRO_MOSAIC.atn file with 5 actions
  • 1 RETRO_MOSAIC.abr file
  • 1 PDF Help File
  • Video link tutorial

Install:

Load the Retro Mosaic Actions

  • RETRO_MOSAIC.atn

Load the Retro Mosaic Tiles Brushes

  • RETRO_MOSAIC.abr

Instructions:

  • Open the image
  • Choose a foreground color for the darker tiles
  • Choose a background color for the lighter tiles
  • From the Actions Panel select the action according to the size
  • Play the action
  • The action ends with a New Layer Group containing all the 12 tile groups

Custom:

  • Each tile group contain 4 masked Color Fill Layers, from bottom to top:
    • Background Color Layer: double click the layer thumbnail to change the joint color
    • Mosaic Lights Layer: double click the layer thumbnail to change the lighter tiles color
    • Mosaic Shadows Layer: double click the layer thumbnail to change the darker tiles color
    • Bevel Effect Layer: show or hide this layer to see the tiles with or without bevel
